What is bulk registration? How can I help?

Bulk registration is a module that allows NGO's / individuals who wish to help us to upload details of several volunteers simultaneously. This is done using a profarma as registration sheet. Please Download from this site, fill the details and post it to the below mentioned address. You can also send the sheet via mail to info@Hyderabadblooddonors.org We shall upload the same to the database. This module enables several organizations to join us in our effort to build a comprehensive directory that can serve as a lifeline to hundreds.
